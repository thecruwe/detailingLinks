# Сайт-визитка для детейлинг-студии + QR-код

Одностраничный сайт (как Linktree) со всеми соцсетями студии. На него ведёт QR-код.

## Что внутри

| Файл | Назначение |
|------|------------|
| `index.html` | Сама страница-визитка. Открывается двойным кликом, работает офлайн. |
| `qr-code.html` | Генератор QR-кода → скачивание PNG (работает в браузере). |
| `README.md` | Эта инструкция. |

## 1. Как посмотреть сайт

Двойной клик по `index.html` — откроется в браузере.

## 2. Где менять данные (в `index.html`)

Ищите по тексту `<!-- ЗАМЕНИТЬ -->`. Кратко:

- **Название и слоган** — блок `<header>` (строки с `GLOSS DETAILING` и «детейлинг студия»).
- **Логотип** — сейчас заглушка с инициалами `GD`. Чтобы вставить свой:
  1. Положите файл (например `logo.png`) рядом с `index.html`.
  2. В стилях включите `.logo img { display:block; }` и добавьте `<img src="logo.png" alt="">` в блок `.logo`.
- **Ссылки** — атрибут `href` у каждого `<a class="link">`:

  | Соцсеть | Что подставить в `href` |
  |---------|------------------------|
  | ВКонтакте | `https://vk.com/ваша_группа` |
  | Instagram | `https://instagram.com/ваш_логин` |
  | Telegram | `https://t.me/ваш_юзернейм` |
  | WhatsApp | `https://wa.me/7XXXXXXXXXX` (только цифры, с `7`) |
  | Звонок | `tel:+70000000000` (со знаком `+`, без пробелов) |
  | Карта | ссылка на Яндекс.Карты с адресом |

- **Подписи** — текст внутри `<span class="sub">…</span>` («Фото работ», «Написать в чат» и т.д.) можно менять свободно.

## 3. Как получить QR-код

1. Разместите `index.html` в интернете (хостинг) и узнайте его адрес, например `https://ваша-студия.ru`.
2. Откройте `qr-code.html` в браузере.
3. Вставьте адрес сайта → QR сгенерируется автоматически.
4. Нажмите **«Скачать QR-код (PNG)»**.

QR кодирует ровно тот адрес, что вы ввели. Смените адрес — перегенерируйте QR.

## 4. Как разместить сайт в интернете (бесплатно)

- **Netlify Drop** — `app.netlify.com/drop`, перетащите папку с `index.html`. Получите адрес вида `имя.netlify.app`.
- **GitHub Pages** — загрузите `index.html` в репозиторий, включите Pages в настройках.
- **Свой домен** — привязывается к любому из вариантов выше.

⚠️ QR-код делайте **после** того, как сайт уже выложен и адрес известен.
